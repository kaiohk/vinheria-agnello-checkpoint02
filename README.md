# Vinheria Agnello - Projeto Checkpoint 02 (Web Development)

## Descrição

Sistema básico para cadastro e análise de vinhos artesanais da Vinheria Agnello. O projeto visa facilitar o gerenciamento da produção, permitindo o registro detalhado de cada lote e a análise técnica das características dos vinhos produzidos.

## 🛠️ O que foi implementado

- **Cadastro Múltiplo:** Uso de repetição para cadastrar vários vinhos até que o usuário decida parar.
- **Funções Reutilizáveis:** Modularização para validar entradas, verificar estoque baixo (< 5), classificar o vinho (jovem/amadurecido/antigo) e exibir alertas.
- **Resumo de Dados:** Exibição final do total de cadastros, vinhos com estoque baixo e identificação da safra mais antiga.
- **Interação:** Sistema operado via `prompt()` e `alert()`, com logs detalhados no console.
